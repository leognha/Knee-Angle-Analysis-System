# Knee-Angle-Analysis-System
智能膝關節角度分析系統

利用多軸感測器JY901M_485來進行資料接收，紀錄受測者在進行膝關節復建時的多軸資訊，利用其中的姿態角度來計算膝關節角度，並顯示在整段復健過程中膝關節活動範圍的最大最小角度，以及透過使用者的輸入能夠判斷本次復健時是否要達到理想的目標角度，在系統的下方也會顯示可視化的數據提供使用者觀看。
